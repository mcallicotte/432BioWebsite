Why Am I Taking Accessible Computing?

I took Human Computer Interaction last semester, and I really enjoyed it. One of my biggest takeaways from that class was that accessible design is required for good design. I want to learn more about accessable computing so that I can use this knowledge to improve my designs.


Website Font

You might have noticed the font on this website is a bit unique. That is because I am using a font called Atkinson Hyperlegible, a font created by the Braille Institute for increased ledgibility for people with low vison.

I used this website as a guide for how to add custom fonts.
